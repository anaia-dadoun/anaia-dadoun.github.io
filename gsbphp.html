Projet GSB PHP

Lors de cette 2 éme année, nous avons appris à utiliser le Modèle Vue Controleur pour les projets php avec notre professeur de développement Web, à travers divers projets. Notre professeur nous a demandé de créer un site pour une gestion de matériel médicales. Ce dernier est composé de plusieurs fonctionalités répartit en deux catégories : les visiteurs et la gestion de matériels. Pour les visiteurs et la gestion de matériels, les fonctions demandées sont : lister, modifier, ajouter, rechercher et supprimer. Pour la gestion des matériels, une fonction réserver et restituer étaient en plus. Nous avons établi ce projet en équipe, pour se partager les différentes tâches, nous avons créer un gestionnaire de projet avec le Logiciel Trello.





Pour ce projet, nous avons commencé par créer une base de données. En premier lieu, nous avons crée un MCD avec le logiciel Looping puis sur Mysql.



La partie du projet concernant les visiteurs représente diverses actions qu'on a dû réaliser. Tout d'abord, pouvoir les lister, ajouter un utilisateur ou bien en modifier un. Toutes ces méthodes ont dû être réalisées à travers un MVC qui redirige les requêtes SQL que j'ai dû créer à la Vue designée pour l'affichage demandé, ou encore vérifier si un visiteur existe déjà lors de sa création.



Pour ajouter un visiteur, on a créé un formulaire avec toutes les informations nécessaires. Une fois que le visiteur a bien été ajouté, il s'affichera directement à la suite de la liste des visiteurs. Pour modifier les coordonnées d'un visiteur, il y aura le même formulaire.
